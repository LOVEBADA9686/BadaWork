2Tier, 3Tier
----------------------

2Tier : C/S : Client / Server
클라이언트에서 비즈니스 로직을 정하고 
데이터베이스에 저장하는 형태 

클라이언트 - 데이터베이스 

비즈니스 로직 : 자바로 데이터베이스를 연동하는 코드들 

3Tier : Web : Client / Server
클아이언트가 미들웨어로 메세지를 주고 받으면서 
데이터베이스에 저장하는 형태 

클라이언트 - 미들웨어 - 데이터베이스 
프론트    - 비즈니스 - 데이터베이스
